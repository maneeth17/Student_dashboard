Important Security Notes
  • JWT token required for all `/api/students/**` routes
  • `POST/PUT/DELETE` allowed for `ADMIN` role only
  • Passwords stored with BCrypt encoding
  • CORS controlled via `APP_CORS_ALLOWED_ORIGINS`

API Quick Summary
  • `POST /auth/register` -> Create user
  • `POST /auth/login` -> Returns JWT token
  • `GET /api/students` -> List students
  • `GET /api/students/average-attendance` -> KPI
  • `PUT/DELETE /api/students/{id}` -> Admin actions
